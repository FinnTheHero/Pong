# Pong

To run Pong:
1. clone and cd into the project
    ```shell
    git clone -b linux-branch https://github.com/FinnTheHero/Pong.git && cd Pong
    ```

2. Create directories `bin` and `obj`:
    ```shell
    mkdir bin obj
    ```

3. Make `build.sh` executable and run it
    ```shell
    chmod +x build.sh && ./build.sh
    ```
